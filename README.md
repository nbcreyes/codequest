# CodeQuest

Story-driven Python education platform for kids aged 9-12.

## Project Structure

codequest/
  client/   React + Vite frontend
  server/   Node + Express backend

## Getting Started

### Prerequisites
- Node.js 18+
- MongoDB Atlas account
- Gmail account for SMTP
- Judge0 RapidAPI key
- Ollama installed locally

### Client setup
cd client
npm install
cp .env.example .env
npm run dev

### Server setup
cd server
npm install
cp .env.example .env
# Fill in your real values in .env
npm run dev
